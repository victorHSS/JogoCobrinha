# Jogo da Cobrinha

Jogo da Cobrinha desenvolvido em allegro (versão inferior a 4) como trabalho da disciplina de Estruturas de Dados da faculdade.

Para rodar no Linux, faça:

```shell
./make
./cobrinha
```
